# MLP-MNIST
A multilayer perceptron to recognize handwritten digits using the **MNIST** dataset.

My hobby project for studying machine learning and neural networks.

To run:
- Install all dependencies using ```pip``` like **NumPy**, **Pandas**, etc.
- If you want to train the network with the training dataset, run ```python3 main.py```.
- If you want to test the network with the testing dataset, run ```python3 test.py```.
- If you want to draw a digit and have the network guess, run ```python3 app.py```.
